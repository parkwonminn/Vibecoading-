# CLAUDE.md — SnapCal

## 프로젝트 개요
- **앱 이름**: SnapCal (스냅칼) — 사진 한 장으로 칼로리를 계산하는 앱
- **저장소**: https://github.com/parkwonminn/Vibecoading-
- **주 브랜치**: `main`
- **플랫폼**: 크로스 플랫폼 (iOS / Android / Web)

## 기술 스택
- **Framework**: React Native + Expo (크로스 플랫폼)
- **언어**: TypeScript
- **로컬 저장소**: AsyncStorage (기기 내 저장)
- **AI 분석**: Claude Vision API (음식 사진 → 칼로리 추정)

## 핵심 기능
1. 카메라로 음식 사진 촬영 또는 갤러리에서 선택
2. AI가 음식 이미지 분석 → 음식명 + 칼로리 추정값 반환
3. 결과를 로컬에 기록 (날짜별 섭취 칼로리 히스토리)

## 제약 조건
- 로그인/회원가입 기능 없음 (익명 로컬 사용)
- 모든 데이터는 기기 로컬에만 저장 (서버 전송 없음)
- 색상: 무채색 기반 UI, 포인트 컬러는 `#FF6B35` (오렌지) 한 가지만 사용

## 디자인 원칙
- 배경: `#FFFFFF` / `#F5F5F5`
- 텍스트: `#1A1A1A` / `#757575`
- 포인트: `#FF6B35` (버튼, 강조 수치에만 사용)
- 컴포넌트: 심플하고 미니멀한 카드형 UI

## 완료 기준 (MVP)
- [ ] 카메라/갤러리에서 음식 사진 입력
- [ ] AI 분석 결과 (음식명, 칼로리 kcal) 화면에 표시
- [ ] 분석 결과 로컬 저장 및 히스토리 조회
