## 설치 방법

### 1. 저장소 클론
```bash
git clone https://github.com/parkwonminn/Vibecoading-
cd Vibecoading-
```

### 2. 의존성 설치
```bash
npm install
# 또는
yarn install
```

### 3. 환경 설정 (필요한 경우)
```bash
# .env 파일이 필요한 경우 복사
cp .env.example .env

# 필요한 환경 변수 설정
```

## 실행 방법

### 개발 모드 실행
```bash
npm run dev
# 또는
yarn dev
```

### 프로덕션 빌드
```bash
npm run build
# 또는
yarn build
```

### 프로덕션 모드 실행
```bash
npm start
# 또는
yarn start
```
